# Open source by Jules Ivanic

Scala libraries, sbt plugins, and a few miscellaneous tools.

## Projects

### Scala libraries

- **[csvzen](https://github.com/guizmaii-opensource/csvzen)** — CSV made fast and simple.
- **[scala-nimbus-jose-jwt](https://github.com/guizmaii-opensource/scala-nimbus-jose-jwt)** — JWT validation for Scala.
- **[zio-uuid](https://github.com/guizmaii-opensource/zio-uuid)** — UUID v1, v6, v7 and TypeID generation with ZIO.
- **[zio-aes](https://github.com/guizmaii-opensource/zio-aes)** — A ZIO `AES` service exposing two functions: `::encrypt` and `::decrypt`.
- **[zio-temporal](https://github.com/guizmaii-opensource/zio-temporal)** — Build invincible apps with ZIO on top of [Temporal](https://temporal.io).

### sbt plugins

- **[sbt-datadog](https://github.com/guizmaii-opensource/sbt-datadog)** — Add the Datadog Java APM agent to your sbt project.
- **[sbt-publish-with-headers](https://github.com/guizmaii-opensource/sbt-publish-with-headers)** — sbt plugin to add custom HTTP headers when publishing Maven artifacts.

### Other

- **[JRubyConcurrentConstantMemoryExcel](https://github.com/guizmaii-opensource/JRubyConcurrentConstantMemoryExcel)** — Write Excel files as fast as possible with constant RAM use in JRuby programs.
